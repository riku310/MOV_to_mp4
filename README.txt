このスクリプトは、ディレクトリ内のすべてのMOVファイルをMP4形式に変換するための手順を示しています。具体的には以下の処理を行います：

1. `dir_input`に入力ディレクトリ、`dir_output`に出力ディレクトリを指定します。
2. 出力ディレクトリ（`output`）を作成します。
3. 入力ディレクトリ（`input`）に移動し、すべてのMOVファイルのリストを取得します。
4. 各MOVファイルに対して以下を行います：
    - 入力ファイル名と出力ファイル名を設定します（拡張子を.MOVから.mp4に変更）。
    - `ffmpeg`コマンドを使用してMOVファイルをMP4形式に変換します。

このスクリプトは、`ffmpeg`ツールを使用してビデオファイルのフォーマットを変換するシェルスクリプトです。
